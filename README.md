# Antiper Henry's Portfolio

A modern, responsive portfolio website built with React, showcasing projects, skills, and contact information.

## Features

- ✨ Smooth scroll animations and interactions
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Modern dark theme with accent colors
- ⚡ Fast and lightweight
- ♿ Accessible HTML structure

## Tech Stack

- **Frontend**: React, CSS-in-JS
- **Build Tool**: Create React App
- **Hosting**: GitHub Pages
- **Fonts**: Google Fonts (Raleway, Fira Code)

## Setup & Development

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Opens [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

### Deploy to GitHub Pages

```bash
npm run deploy
```

This will build the project and deploy it to GitHub Pages.

## File Structure

```
github-portfolio/
├── public/
│   └── index.html          # HTML template
├── src/
│   ├── index.js            # React entry point
│   └── Portfolio.jsx       # Main portfolio component
├── package.json            # Dependencies and scripts
└── README.md              # This file
```

## Customization

Edit `src/Portfolio.jsx` to update:
- Personal information (name, email, phone)
- Navigation links
- Skills and project details
- Colors and styling (CSS variables in `globalStyles`)
- Social media links
